## Pre-requisitos
- Java 11
- Docker Compose

## Hecho con
- Kotlin (1.6.0)
- Spring Boot (2.6.0)
- Gradle (7.2)

## Build

Con una terminal abierta en la raíz del proyecto, ejecutar el comando:

```bash
./gradlew build
```

## Tests

Con una terminal abierta en la raíz del proyecto, ejecutar el comando:

```bash
./gradlew test
```

## Levantando la app

Con una terminal abierta en la raíz del proyecto, ejecutar el comando:

```bash
sudo docker-compose up --build -d
```
